# SAP Business One DI API DLLs

Copiez les fichiers suivants depuis votre installation SAP Business One :

```
C:\Program Files (x86)\SAP\SAP Business One DI API\
```

Fichiers requis :
- `SAPbobsCOM.dll` (obligatoire)
- `SAPbobsCOM.xml` (documentation IntelliSense, optionnel)

⚠️ **Important** : La DI API SAP est 32-bit, donc le projet doit être compilé en x86.
